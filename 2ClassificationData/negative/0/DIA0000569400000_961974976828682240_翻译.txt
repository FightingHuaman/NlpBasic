180天体重管理训练营--企业员工的健康计划
对象：全体员工
时间：180天
规模：20-50人
要求：1.设备要求：投影、电脑外接音频线、
      无线话筒、瑜伽垫
      2.场地要求：可容纳20-50人活动的空间

引言：
提起企业员工，尤其是办公室白领一族，常常会和自由，潇洒，独立，自信联系在一起，然而在光鲜亮丽的背后，其实潜藏着许多我们想象不到的健康问题，尤其是由久坐少动导致的肥胖，从而引发三高，颈椎病，腰间盘突出等问题......本计划以地面训练+线上课程+每日监督的形式，帮助员工养成良好的健康习惯，帮助员工改善身体机能，强健体魄，改善体型，对于身姿体态，三高，颈椎腰椎问题都能起到较为明显的改善作用。

培训收益：
?训练对企业的好处：
1、 使员工明确并认同组织的目标，增强企业的凝聚力
2、 树立相互配合，相互支持的团队意识。
3、 改善企业间的人际关系，形成积极向上的组织氛围。
4、 使员工表现出更佳的领导与管理才能。
5、 挖掘员工的内在能量，使之聚合，并在工作中全部释放出来。

?训练对员工的好处：
1、 改善身体机能，强健体魄。
2、 认识自身潜能，增强自信心，改进自身形象。
3、 克服心理惰性，完善性格结构。磨炼战胜困难的意志。
4、 调适身心、不浮躁、不颓废，更好的面对工作与生活的挑战。
5、 认识群体的作用，增进对集体的参与意识和责任心。
6、 启发想象力与创造性，提高解决问题的能力。
7、 学习欣赏别人，学会关心别人。
8、 助人为乐，关爱生命和自然。
9、 情感沟通和表达能力增强，人际关系趋向和谐。

课程纲要:
第一节课程：训练营开营
1. 课程动员+健康塑身基础营养课程（约30分钟，学会日常如何控糖、如何正确饮食）
2. 体态评估+分组（约用时30分钟，教会个人了解自身的身体健康现状，根据不同体态分成不同的小组）
3. 健身+营养答疑+课后作业（约30分钟）
4. 组建微信群（教练管理，每日打卡监督，在固定时间答疑）
第二节课程（知识课堂）：饮食与营养理念导入
中医专家将通过详细的阐述食物的寒热属性，烹调方法的利弊，并结合不同体质人的不同需求特点，给出合理的饮食方案，在改善体质的同时不知不觉达到了获得理想体重的目的。以线上微课的形式进行，可面向全体员工与家人一起参与，为健康塑身营造良好的饮食氛围。
第三节：实践课堂之塑身训练（一）
1. 热身（约10分钟）
2. 基础自重训练（约40分钟，轻强度刺激，为下次训练做准备）
3. 静态拉伸（约40分钟，教会个人为什么要拉伸、如何正确拉伸，并学会如何给他人拉伸）
第四节：
1. 了解、更正上一次训练课中和课后问题，并答疑（约30分钟）
2. 热身（约10分钟）
3. 全身律动（约40分钟，培养耐力、交叉训练、下肢力量、快速燃脂）
4. 静态拉伸（约10分钟）
第五节：体重与健康知识巩固
医学专家将在线上与员工分享体重管理与健康的知识，过重或过轻都让人苦恼，为了美丽，也为了健康，体重的控制都是无可厚非的。体重管理，是健康管理的第一步；而健康，是身体、心理、社会关系的动态平衡。项目中期二次动员，以线上微课形式进行，可邀请家人和更多的员工加入，帮助企业形成良好的健康氛围。

2. 热身（约10分钟）
3. 拳手终结（约40分钟，腹部核心力量）
4. 静态拉伸（约10分钟）
第七节
2. 热身（约10分钟）
3. 力量融合（约40分钟，胸部、肩部力量，全身力量，力量发展）
4. 静态拉伸（约10分钟）

2. 热身（约10分钟）
3. 三体重训（约40分钟，背部和手臂力量，核心肌群稳定性、加速新陈代谢，下肢力量）
4. 静态拉伸（约10分钟）
第九节：训练营结业典礼

** 180天训练营期间，教练将在微信群中陪伴所有学员一同进步，发布每日训练内容，督促监督学员完成每日训练，答疑解惑。
