Introduction 
Whole view model 是帮助企业进行商业创新的指导工具，其模型以4w框架（what，who，why，how）为核心，配合相关的创新流程，创新方法，从用户需求，技术能力，商业价值，产品形态四个角度帮助创新者系统全面的思考创新活动，从而创造出具有市场价值的切实可行的创新方案。但随着社会大众对企业的社会责任和环保责任要求越来越高，企业创新不应该仅仅考虑商业层面的价值，还要考虑环境层面和社会层面的效益。因此，原有的whole view模型在指导兼顾社会价值和环境价值的创新时，就产生了一些局限性：

1. 线性的创新流程，缺乏灵活性， whole view模型基于4W框架（what，who，why，how），创新者通过使用模型中提到的工具与方法，依序思考及解答与4W相关的问题，从而实现创新活动，这种按特定路线发展的创新流程缺乏灵活性，容易造成思维局限
2. 在why模块中，模型仅关注创新产生的经济价值，忽视了其可能产生的社会价值和环境价值，这样的创新也有可能在经济成功的同时带来环境破坏或负面的社会影响。
3. 在who模块中，模型仅关注核心利益相关者（stakehoders），忽略了对其他外围利益相关者.事实上任何一个创新成果不仅会对系统内的人产生直接影响（用户，生产者，服务者等），也会对系统外的人，环境或社会产生或多或少的间接影响，如工业化对全球变暖的潜在影响。
4. 在what模块中，仅关注了具体解决方案的实现（solution），忽略了对方案可持续性的评估以及对产品生命周期的考量。
5. 在how模块中，对于创新实现的可能性的评估主要集中在对技术限制和盈利限制的考量上，忽视了可能存在的环境限制和社会限制因素。

从上面的分析可以看出，whole view model可以帮助企业（organization）在经济活动层面系统化的进行创新（如何了解用户需求，了解技术限制，如何创造经济价值），但在社会和环境层面，该模型存在一定的功能缺失。
Triple layered whole view model
指导一个完整的创新活动需要创造一个超越经济中心（economic foucus）的模型，即以whole view model 为基础，基于4W框架，创造一个新的包含环境层面和社会层面的创新模型（图1）。

原有的模型在经济层面很好的解释了用户，技术，商业和产品间的关系。通过思考以下四个问题，企业有机会创造出符合市场要求同时又能实现商业利润的创新，服务，或者着产品 
What should we make？
Who will want it?
Why will it create enough value to make it worth producing?
How should we organize our key activities so we can make it?

然而，如果我们将社会层面和环境层面的因素考虑进去，这四个问题就显得过于片面了，为了创造新的三层全景框架，我们需要在社会层面思考下面四个问题：
What我们做的东西对社会是否产生正面或负面影响？
Who哪些社会阶层会因为这个创新受益或受害？
Why为什么这一创新会获得社会大众或社会道德的支持或反对？
How公司如何利用社会影响实现创新或摆脱社会压力？
在环境层面，我们应该思考以下四个问题：
What我们做的东西对环境是否产生正面或负面影响？
Who哪些自然环境或自然资源会因为这个创新受益或受害？
Why为什么这一创新会获得环保主义者的支持或反对？
How公司如何彰显自身环保责任实现创新或摆脱环保压力？

New dynamics in a layered approach to a triple layered whole view model
不同于过去单一层面，线性的创新流程。由于加入了社会层面和环境层面两个layers，应用新模型的创新流程会更加灵活与多变，其主要存在两种形式：horizontal coherence和vertical coherence。
在horizontal coherence中，整体的创新流程是螺旋式上升的模式，创新者可以按原有的whole view模型的思考方式和创新流程对每个层面分别进行深入的思考，然后将下一层面的思考结论代入上一层面作为前提条件。如图所示（图2），创新者在社会层面分别思考了what，why，who，how四个问题后，得出了一个企业创新的大体原则，然后将这一原则带入环保层面进行新一轮的思考，经过这样的创新流程，企业的创新成果会被限定在一个能被社会，环境和商业三方面接受的范围内，最大化的实现多重价值。

在vertical coherence模式则不容，对what，why，who，how四个问题是分别进行跨层面深入思考的，如图所示(图3)，当创新者思考"who"这个问题时，需要同时考虑到用户，社会大众和环保主义者。这一模式不会将创新者的创新范围进行逐步限缩，而是在一开始就以一种统筹兼顾的模式进行思考，使得创新流程更加整体，缺点是容易在经济，环境和社会利益出现矛盾时难以取舍。

结论
随着越来越多的企业开始重视自身的社会责任和环保责任，创造一个适用于这一趋势的创新框架是必要的。新的三层全景框架是为应对这一趋势而做的一次初步探索，讨论了在原有框架的基础上，通过加入环境和社会两个层面，促进创新者更全面的思考创新对商业，社会，环境三个层面的影响。但是这一新的三层框架只是一个初步的设想，相对于原有框架成熟的创新流程以及与之配套的丰富的创新方法于工具，新的模型在社会层和环保层上还缺乏与之对应的创新工具，在未来的研究中，我们应该以这一新框架为基础，寻找或创造与之配套的创新工具，并充分验证这一新框架的可行性。
